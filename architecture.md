# CSX-IPTV2 Architecture

## Overview
An Android plugin built for the Cloudstream ecosystem.

## Technology Stack
- **Language**: Kotlin
- **Build System**: Gradle (Kotlin DSL)
- **Framework**: Cloudstream Plugin API

## Responsibilities
- Integrates the IPTV directory natively into the Cloudstream Android app.
- Allows users to browse categories, discover live links, and play streams utilizing the overarching proxy network without relying on the web frontend.
